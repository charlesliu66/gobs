# TASK-C：前端"待确认标签"与检索体验

## 目标
让用户 5 分钟内把混乱素材变成可用库。

## 页面
| 组件 | 说明 |
|------|------|
| `AssetLibraryPage` | 资产总览（入口页） |
| `AssetImportPanel` | 导入任务（进度+重试） |
| `AssetReviewQueue` | 待确认标签（批量审核） |
| `AssetSearchPanel` | 筛选+语义搜索 |

## 核心交互
- 批量打标（勾选多项 → 统一加/改标签）
- 一键应用到相似素材（按 AI 相似度匹配）
- 快速进入生成/剪辑入口（每个素材卡片有"用于生成""用于剪辑"按钮）

## 验收标准
- [ ] 待确认项可批量处理（勾选 → 确认/拒绝/修改）
- [ ] 至少支持 6 维筛选（角色/场景/类型/用途/比例/时长）
- [ ] 点击素材可一键"用于生成"（跳转 Studio 并带入素材引用）
