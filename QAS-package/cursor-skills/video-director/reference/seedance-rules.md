# Seedance / 剪映 平台内容规则

基于实际审核调整经验整理。出现「你输入的文字不符合平台规则」时，多为以下类型内容触发。

---

## @ 图片引用语法（必须遵守）

Seedance/剪映 支持在 prompt 中用 `@图片N` 引用上传的素材，上传顺序决定编号。

| 上传顺序 | 引用方式 | 说明 |
|----------|----------|------|
| 第 1 个 | `@图片1` | 对应第一个上传的图片/视频 |
| 第 2 个 | `@图片2` | 对应第二个 |
| 第 3 个 | `@图片3` | 以此类推 |

### 写法示例

- **指定形象**：`@图片1 作为浪人形象，在雨夜竹林中挥刀`
- **指定动作**：`@图片2 在雨中缓慢行走`
- **首尾帧**：`@图片1 作为首帧，@图片2 作为尾帧，模仿中间动作`
- **多图组合**：`@图片1 与 @图片2 遥相对望，雨幕朦胧`

### 规则

1. 有上传素材时，prompt **必须**用 `@图片N` 引用，N 与**上传顺序**一一对应
2. **映射以用户指定为准**：若用户说「浪人要写成 @图片2」，则浪人 = `@图片2`，怪物 = `@图片1`
3. 引用时写清动作/场景，如：`@图片2 在雨夜竹林中挥刀` 而非仅写 `@图片2`

### 典型映射示例（文件夹内有 怪物.png、浪人.png）

| 文件名   | 引用   | 说明                       |
|----------|--------|----------------------------|
| 怪物.png | @图片1 | 第一个上传（按文件顺序）   |
| 浪人.png | @图片2 | 第二个上传                 |

- 提及浪人时 → 写 `@图片2`
- 提及怪物/对手时 → 写 `@图片1`（或按平台规则改用「神秘对手」等表述）

## 自动映射（按素材文件名）

**规则**：从文件名提取语义标签（去掉扩展名），在 prompt 中**自动替换**对应提及。

| 步骤 | 操作 |
|------|------|
| 1 | 扫描文件夹，得到 `[怪物.png, 浪人.png]` |
| 2 | 提取标签：`怪物`、`浪人`，按文件名排序确定上传顺序 → 怪物=@图片1，浪人=@图片2 |
| 3 | 生成 prompt 时，凡提及「浪人」→ 替换为 `@图片2`，提及「怪物」→ 替换为 `@图片1` |

**文件名→标签**：`浪人.png` → `浪人`，`怪物.webp` → `怪物`（仅保留主文件名，去掉扩展名）。

**优先顺序**：用户明确指定（如「浪人=@图片2」）> 自动按文件名映射。

## 禁用/高风险词汇（必须替换或删除）

### 暴力/斩杀类
| ❌ 禁止 | ✅ 替换为 |
|--------|----------|
| 斩杀、砍杀、劈斩、斩落、劈砍 | 挥刀、收刀、刀光划过 |
| 击杀、消灭、击倒 | 拉开距离、转身离去、收势 |
| 倒地、倒下、死亡 | 退让、遥相对望、错身而过 |
| 血、流血、血迹 | 删除或改为「氛围」「张力」 |

### 生物/怪物类
| ❌ 禁止 | ✅ 替换为 |
|--------|----------|
| 怪物、巨兽、妖兽 | 神秘对手、大型生物、对方、身影 |
| 嘶吼、咆哮、怒吼 | 威慑姿态、缓慢移动、遥相对望 |
| 扑来、扑击、猛扑 | 逼近、靠近、错身 |
| 利爪、獠牙 | 删除或改为抽象描述 |

### 武器/对抗类（视平台敏感度）
| ⚠️ 慎用 | ✅ 更安全 |
|---------|----------|
| 格挡、反击、还击 | 闪避、挥刀、收势 |
| 刀剑碰撞、火星四溅 | 刀光剑影、身形交错 |
| 交锋、对决、厮杀 | 起舞、练刀、遥相对望 |

---

## 安全化策略

### 策略 A：弱化对抗
- 将「A 打 B」改为「A 与 B 遥相对望 / 交错而过」
- 将「战胜」改为「收刀离去 / 拉开距离」

### 策略 B：抽象化对手
- 「怪物」→「神秘生物」「对手」「身影」「轮廓」
- 去掉生物特征词（嘶吼、利爪、獠牙）

### 策略 C：改为独舞/意境
- 去掉所有「对手」，保留单人动作
- 例：浪人打怪物 → 浪人在雨夜竹林中练刀、收刀、远眺

### 策略 D：删除结果描述
- 不写「谁倒下」「谁胜利」
- 以「定格」「剪影」「雨幕朦胧」等氛围收尾

## 审核清单（执行 Step 3 时逐项检查）

- [ ] 无 斩杀/砍杀/劈斩/击杀 等词
- [ ] 无 怪物/巨兽/妖兽
- [ ] 无 嘶吼/咆哮/扑来/利爪/獠牙
- [ ] 无 血/流血/倒地/死亡
- [ ] 对抗性动作已弱化为 闪避/挥刀/收势/遥相对望
- [ ] 若有「对手」，已改为 对方/神秘生物/身影

## 示例：合规前后对比

**原文**：怪物嘶吼扑来，浪人斩落，怪物倒地。

**合规后**：对方逼近，浪人挥刀闪避，双方拉开距离，浪人收刀而立。

## Seedance 2.0 扩展规范（2026 终极版）

以下内容源于即梦 Seedance 2.0 提示词安全规范，用于最大化审核通过率 (>95%)。

### 绝对禁区 (Red Zone) — 触发即拦截

| 类别 | 禁忌词示例 | 安全替代 |
|------|------------|----------|
| **真人/深假** | 真人照片、换脸、Deepfake、还原本人、真实人脸 | 3D 渲染角色、数字艺术形象、动画风格人物、CGI 角色 |
| **政治/敏感** | 领导人、政党、抗议、示威、恐怖组织 | 虚构城市、幻想世界、戏剧化冲突 |
| **色情/低俗** | 色情、裸体、性暗示、诱惑姿势、内衣秀 | 优雅着装、时尚造型、得体服饰 |
| **暴力/血腥** | 血腥、断肢、杀戮、自杀、酷刑、尸体 | 激烈动作、紧张对峙、戏剧冲突、暗调光影 |
| **违法/危险** | 吸毒、赌博、诈骗、制造武器 | 移除或改为正面/中性场景 |

### 版权与 IP 限制 (Yellow Zone)

| 类型 | 限制词 | 安全替代 |
|------|--------|----------|
| **影视 IP** | 迪士尼、漫威、哈利波特、宫崎骏、奥特曼 | 童话风格、科幻风格、日式手绘动画风、复古特摄风 |
| **品牌** | Nike、Apple、可口可乐 | 知名运动品牌、极简智能手机、红色罐装饮料 |
| **公众人物** | 明星姓名、政治家 | 外貌特征描述，如「亚洲中年男性，功夫巨星气质」 |

### 魔法词与安全声明（必须添加）

**开头声明**（任选其一或组合）：
- `【原创数字艺术作品】`、`【积极正向主题】`、`【3D 渲染风格】`

**结尾声明**（任选其一或组合）：
- `，适合全年龄段观看`、`，无版权争议`、`，无敏感元素`、`，虚构场景`、`，电影级特效`

### 高危词一键替换表

| 原高危词 | 替换为 (安全词) |
|----------|-----------------|
| 真人/真实 | 3D 渲染/数字艺术/CGI/动画风格 |
| 明星姓名 | [发色]+[脸型]+[气质] 描述 |
| 迪士尼/漫威 | 童话风格/科幻风格/动画电影质感 |
| 性感/诱惑 | 优雅/时尚/精致/魅力 |
| 血腥/暴力 | 激烈/紧张/戏剧化/动作感 |
| 恐怖/鬼魂 | 神秘/悬疑/暗调/氛围感 |
| 死亡/尸体 | 沉睡/静止/落幕/结束 |
| 品牌名 | 通用描述（如「运动品牌」） |
| 换脸/克隆 | 形象转换/角色变身/风格变化 |

### 技术限制（避免无效指令）

- **无效**：生成 2 小时长视频、8K 原片、精确控制每一帧、完全一致角色（跨视频）
- **事实**：单段视频最长约 15 秒，分辨率由平台设定
- **负面提示**：平台支持有限，应在正向提示词中明确正确内容；可用负面词：模糊、低质量、畸形、多余肢体、水印、字幕、变形

### 提交前自查（5 秒快检）

- [ ] 是否包含真实人物姓名/肖像？ → 改为特征描述/3D 角色
- [ ] 是否包含具体 IP/品牌名称？ → 改为风格描述
- [ ] 是否有暴露/性感相关词汇？ → 改为优雅/时尚
- [ ] 是否有暴力/血腥相关词汇？ → 改为动作/戏剧化
- [ ] 是否添加了「安全声明」？ → 开头/结尾至少各一
- [ ] 提示词长度？ → 控制在 500 字以内
- [ ] 是否使用了 @ 引用素材？ → 建议使用合规素材

### 常见拒绝代码及解决

| 拒绝提示 | 解决方案 |
|----------|----------|
| 内容涉及真人形象 | 改为「3D 角色」「动画人物」 |
| 涉及版权内容 | 删除具体名称，用风格替代 |
| 内容不适合展示 | 删除暴露、暴力相关描述 |
| 提示词包含敏感词 | 对照高危词表逐一替换 |
| 无法识别的内容 | 简化描述，明确主体和动作 |
| 素材引用违规 | 使用官方素材库或自有合规素材 |
