# 剪辑 Agent 使用指南

> 面向 QAS 视频剪辑工作台的 AI 剪辑 Agent。
> 本指南基于代码真实行为（`editorAgentService.ts` / `editorHighlightCandidates.ts` / `bgmIntent.ts`）整理，不是空洞说教。

---

## 1. Agent 识别哪几类关键词

你每条指令会被 5 个模块同时扫描。命中任何一个都会让 Agent 按对应策略剪片：

| 识别项 | 触发关键词 | 效果 |
|---|---|---|
| **战斗意图** | `战斗 / 打斗 / 高光 / 精彩 / BOSS / 击杀 / 连招 / 对决 / 激战` | 切「快切混剪」叙事；候选窗优先选中段、按"战斗帧分数"排序 |
| **时长** | `10 秒 / 15s / 20 秒的 / 做一个 25 秒` | 设成片目标时长；不写 → 用素材 × 默认 ≈30s，可能和你想的不一样 |
| **BGM 意图** | `配乐 / 背景音乐 / BGM / OST / 高燃音乐 / 来一段音乐 / 加个伴奏` | Agent 剪完**自动**跑 Lyria（或 Suno）生成 BGM 贴到 a2 轨 |
| **AOV/王者** | `王者荣耀 / AOV / 推塔 / 五杀` | 叠一层 AOV DSL 规则（结构/必备事件/比例） |
| **指定角色** | 角色库里已登记的完整角色名（严格匹配，不支持模糊） | 从视觉识别中挑该角色出现的窗口；写错会报错并给建议 |

## 1.5 支持哪些叙事类型（不只战斗混剪！）

代码里实装了 3 套叙事模板，**默认是"角色故事"（宣传片结构），不是战斗模板**：

| 模板 | 触发条件 | 结构 | 典型场景 |
|---|---|---|---|
| **经典高光** | 指令含 `战斗 / 击杀 / BOSS / 连招` | 钩子 → 铺垫 → 高潮 → 收尾 | 战斗混剪、击杀集锦 |
| **角色故事** ← 默认 | 不含战斗词 | 登场 → 能力展示 → 危机 → 逆转 | 角色 PV、皮肤发布、英雄宣传 |
| **节奏混剪** | 有 BGM 且做了节拍分析 | intro → build → drop → outro | 卡点混剪、病毒视频 |

非战斗指令示例：

- **角色 PV**：`做一个 20 秒英雄宣传片，前 5 秒角色特写，10 秒展示技能连招，最后 5 秒反转收尾，配史诗 BGM`
- **皮肤宣传**：`15 秒新皮肤宣传，开头定格特写，中间技能特效，结尾 logo 镜头，配华丽电子 BGM`
- **玩法介绍**：`25 秒玩法宣传，从所选素材挑 4-5 个不同玩法场景快切，配欢快 BGM`
- **剧情向**：`30 秒剧情短片，前 8 秒氛围铺垫 → 中间 15 秒冲突 → 后 7 秒情绪释放，配温暖钢琴 BGM`
- **病毒 / 爆款**：`12 秒 TikTok 竖屏爆款节奏混剪，前 3 秒 Hook，中间快切主体，末尾反转收尾，配洗脑 BGM`
- **氛围 Vlog**：`25 秒治愈氛围片，慢节奏，柔和转场，配氛围钢琴 BGM`
- **卡点混剪**：`20 秒卡节拍混剪，按拍点切镜，配动感电子 BGM`

## 2. 黄金公式（一条指令解决战斗）

```
我要做一个【 X 秒 / 多少秒 】的【 TikTok 竖屏 / 横屏 】【 战斗 / 高光 / 剧情 】混剪，
从所选素材里挑【 快切 / 慢节奏 】画面，配上【 高燃 / 燃曲 / 氛围感 】BGM，
结构：开场钩子 → 主体 → 结尾反转。
```

这套话同时命中 时长 + 战斗意图 + BGM 意图，Plan 模型返回空内容的概率最小。

## 3. 场景示例

### ✅ 推荐写法

**战斗向 TikTok**
> 帮我做一个 15 秒的 TikTok 竖屏战斗高光混剪，从选中素材里挑激烈的战斗画面，开头 3 秒要有钩子镜头，结尾给一个反转，配一首高燃电子 BGM。

**游戏精彩时刻**
> 用所选 5 段素材剪一个 20 秒的击杀连招集锦，按快切节奏走，BGM 要燃。

**剧情叙事**
> 做一个 30 秒剧情向短片：前 5 秒氛围铺垫 → 中间 20 秒冲突 → 最后 5 秒情绪结尾，配温暖感钢琴 BGM。

**指定角色**
> 用素材剪一个「夜影」角色的 15 秒高光混剪。
> *（"夜影"必须在角色库里存在，否则先问"有哪些可选角色？"）*

### ❌ 踩坑写法

| 你写的 | 为什么挂 | 改法 |
|---|---|---|
| 帮我做一份符合 tiktok 风格的剪辑 | 无时长 / 节奏 / BGM 关键词；Plan 模型自由发挥，空返回概率极高 | 加"15 秒"+"战斗或高光"+"配高燃 BGM" |
| 给我剪个战斗 demo | 时长缺失 → 默认 30s | "15 秒战斗混剪" |
| 用李白剪一段 | 角色名和库对不上 → Agent 抛错 | 先问角色列表，或用 Agent 给的建议名 |
| 随便剪一下就行 | 空信息 → 模型瞎猜 → 容易挂 | 至少给 时长 + 氛围 + BGM 三要素 |

## 5. 常见坑 & 解决

### 1）"错误：network error" / "Compass Gemini 返回内容为空"
- 已增强：LLM 空返回时自动重试（会轮换到备用 Key），Plan 阶段失败会自动降级到 Legacy 兜底 prompt。
- 如果仍挂：说明三次都空。最可能是 prompt 触发 Gemini safety filter 或者 maxTokens 不够。把指令改短、去掉"击杀/血腥"等敏感词、减少勾选的素材（推荐 2–6 段）。

### 2）"agentTracks.find is not a function" / "模型输出 JSON 缺少 project 字段"
- 已修：对 Legacy 模型偶发生成的非数组 tracks / subtitles 做 `Array.isArray` 强制归一化。
- 再遇到就是模型真的输出烂了 → 换一条更具体的指令重试。

### 3）"ffmpeg 抽取音频失败 (code 234)"
- 已降级：素材没有音轨时只是跳过音频分析，不会中断剪辑。
- 不用管这条日志，它只影响"BGM 节拍对齐"功能。

### 4）"未找到与参考角色目录完全一致的命名"
- 严格匹配。Agent 会给出建议名，原样贴进指令重发。

### 5）Agent 把 BGM / 字幕清掉了
- 已修（v0.52+ 合并策略）：用户已配置的 BGM / 文字 / 字幕不会被 Agent 覆盖，只有"空轨"才会被 Agent 填充。
- 如果希望 Agent 换 BGM，先自己把 a2 轨删空再跑指令。

### 6）素材越多越好？
- 不是。推荐 2–6 段。<2 发挥不出来，>8 prompt 过长容易触发 LLM 空返回。

## 6. 剪完之后怎么继续调

Agent 剪一遍后，用**第二条指令**微调而不是重新说一遍：

- `把第 3 段加快 1.5 倍`
- `去掉第 2 段`
- `在开头加一段 2 秒的黑场淡入`
- `把字幕换成英文`
- `换一首更紧张的 BGM`（会重新走 BGM 生成）

或者直接用 UI：
- 时间轴上的片段可以拖拽、Trim、Split
- 右下角配乐面板：重试 / 切 Lyria↔Suno / 跳过
- Undo/Redo 已经修过边界 bug，拖拽 / Trim 一次撤销即可完全恢复

---

## 7. Quick Reference

一句话：**"时长 + 氛围/战斗意图 + BGM 关键词"** 是最小可用指令。

例：`用所选素材剪一个 15 秒的战斗高光混剪，配高燃 BGM` ← 这条永远能跑。
