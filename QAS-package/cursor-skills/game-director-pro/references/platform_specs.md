# Platform Specs — TikTok, Reels, Shorts (Game Trailers)

## 通用竖屏规则（TikTok / Reels / Shorts）

- **比例**：竖屏 **9:16** 为主（横屏 16:9 用于 YouTube 主站/预贴片时单独说明）。  
- **时长**：**15–30 秒**为游戏短视频甜点；前 **3 秒**必须完成 Hook，最后 **3–5 秒**留 CTA。  
- **静音兼容**：约 40% 用户静音，前 3s 的文字/画面须在无声音下也能传达信息。

---

## TikTok

| Parameter | Spec |
|-----------|------|
| Aspect Ratio | **9:16 vertical** (mandatory) |
| Resolution | 1080 × 1920 px |
| Optimal Duration | **15–30s** (sweet spot for game content) |
| Max Duration | 60s (short-form); 10min (long-form) |
| File Format | MP4 / MOV |
| Frame Rate | 24–60fps |
| Hook Window | **First 3 seconds** — must stop the scroll |
| CTA Position | Last 3–5 seconds |

### TikTok Algorithm Hooks (Game Content)
- **Immediate action**: Start mid-battle, not with a logo
- **Curiosity gap**: "I've never seen this before" feeling
- **Text overlay**: First 3s must work WITHOUT sound (40% users muted)
- **Fast cuts**: Game trailers perform best at 1.5–3s average shot length
- **Sound-on moments**: Design a "sound on" reward — satisfying SFX, hype music drop
- **Loop-worthy**: Last frame should visually connect back to first frame

### TikTok Content Formats That Work for Games
| Format | Example Hook | Avg. Watch Time |
|--------|-------------|----------------|
| "First 30 seconds of gameplay" | "You won't survive this opening" | High |
| Character reveal | "Meet the character everyone's talking about" | High |
| Mechanic showcase | "This one ability breaks the game" | Very High |
| Before/After (game evolution) | "This game in 2022 vs 2025" | Medium |
| Lore drop | "The real reason [character] betrayed..." | Medium |

## YouTube (Shorts & Pre-roll)

| Parameter | Spec |
|-----------|------|
| Aspect Ratio | **16:9 horizontal** (standard) or 9:16 (Shorts) |
| Resolution | 1920 × 1080 px |
| Optimal Duration | **15–30s** (pre-roll/bumper) or **60s** (Shorts) |
| Hook Window | **First 5 seconds** (pre-roll skip threshold) |
| CTA Position | Last 5 seconds + end screen |

### YouTube Pre-roll Rules
- First 5s = unskippable — **do not waste on logo or loading screen**
- Show the most exciting/surprising moment of the game first
- Include game title text within first 5s
- CTA: "Available now on [platform]" or "Wishlist on Steam"

---

## Instagram Reels / YouTube Shorts（竖屏）

| Parameter | Spec |
|-----------|------|
| Aspect Ratio | **9:16** |
| Optimal Duration | **15–30s**（与 TikTok 一致） |
| Hook | 前 3s 抓人；Reels 偏画质与音乐，Shorts 偏玩法/解说感 |
| CTA | 最后 3–5s，可加「链接在简介」等引导 |

---

## Both Platforms Strategy

When targeting both TikTok + YouTube simultaneously:
1. **Shoot vertical-first (9:16)** — compose for mobile
2. **Safe zone**: Keep key subjects in center 60% of frame for 16:9 crop
3. **Subtitle/text**: Center-align, avoid top/bottom 15% of frame
4. **Duration**: 15–30s works for both
5. Export two versions: 9:16 original + 16:9 cropped
