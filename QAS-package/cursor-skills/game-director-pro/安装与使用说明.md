# Game Director Pro Skill — 安装与使用说明

给**其他人**安装本 skill 时，按以下步骤操作即可。

---

## 一、安装

### 1. 获取 skill 包

- 从分享者处拿到 **整个 `game-director-pro` 文件夹**（或解压 `game-director-pro.zip`）。
- 确保文件夹内包含：`SKILL.md`、`README.md`、`references/`、`scripts/`、`.env.example`。

### 2. 放入 Cursor 的 skills 目录

**方式 A：仅当前项目可用**

- 在你的**项目根目录**下创建 `.cursor/skills/`（若没有则新建）。
- 将 `game-director-pro` 整个文件夹复制到 `.cursor/skills/` 下：
  ```
  你的项目/
  └── .cursor/
      └── skills/
          └── game-director-pro/   ← 整个文件夹放这里
              ├── SKILL.md
              ├── README.md
              ├── 安装与使用说明.md
              ├── references/
              └── scripts/
  ```

**方式 B：全局可用（所有项目）**

- 将 `game-director-pro` 复制到 Cursor 的**用户级 skills 目录**（具体路径见 Cursor 设置中的 Skills 路径，常见为 `~/.cursor/skills/` 或类似位置）。

### 3. 确认生效

- 在 Cursor 中与 AI 对话时，提到「做游戏宣传片」「新英雄视频」「分镜脚本」等，Agent 应会自动加载本 skill 并按流程执行。

---

## 二、配置（按需）

需要用到**首帧/尾帧生图**或**视频生成**时，才配置 API。

### 1. 首帧/尾帧生图（火山 Ark / 其他 API）

- 在**使用本 skill 的项目根目录**下，将 skill 内的 `.env.example` 复制为 `.env`（或在本 skill 目录下复制，视脚本从何处加载 .env）。
- 编辑 `.env`，填入你的 Key（**不要**把 `.env` 提交到 Git）：
  - **火山 Ark 文生图**：在火山方舟控制台创建 Seedream 文生图模型，拿到 Endpoint ID，然后填写：
    ```env
    FRAME_IMAGE_API_KEY=你的Ark的UUID
    FRAME_IMAGE_API_URL=volcano
    FRAME_IMAGE_API_MODEL=ep-xxxxxxxx-xxxx
    ```
  - 其他 API 的配置方式见：`references/frame_image_api_guide.md`。

### 2. 视频生成（火山 Ark Seedance）

- 在 `.env` 中配置：
  ```env
  SEEDANCE_API_KEY=你的Ark的UUID
  ```
- 更多说明见：`references/kling_api_docs.md`。

### 3. 可选 Python 依赖

- 若希望脚本从 `.env` 自动读 Key，可安装：`pip install python-dotenv`
- 不安装也可运行脚本，只需在终端里用环境变量传入 Key。

---

## 三、使用流程（由 Agent 按 SKILL 执行）

1. **Stage 1**：和 AI 说要做什么视频（如「新英雄 XX 宣传片」「新地图展示」），AI 会产出创意简报并请你确认。
2. **Stage 2**：确认后，AI 产出分镜表（含每镜首帧/尾帧描述与提示词），并可按你提供的 JSON 调用脚本生成首帧/尾帧图。
3. **Stage 2.5**：I2V 运镜建议、剪映后期指示。
4. **Stage 3**：确认分镜后，AI 可调用视频生成脚本产出各镜视频（需已配置 SEEDANCE_API_KEY）。
5. **Stage 3.5**：成片合成（需本机安装 FFmpeg）。
6. **Stage 3.6**：社媒发布策略。

**重要**：首帧/尾帧图按**分镜脚本**逐镜生成（每镜不同构图），不要用同一张参考图拉伸到所有镜头。角色一致性靠分镜 JSON 中的「角色设定卡」前缀。

---

## 四、脚本直接调用示例

在**项目根目录**下执行（路径按你实际安装位置调整）：

```bash
# 按分镜 JSON 生成首帧/尾帧图（需已配置生图 API）
python .cursor/skills/game-director-pro/scripts/generate_frame_images.py -i 你的分镜.json -o 输出目录/frames

# 成片合成（需已有所需 shot_xx.mp4 与 FFmpeg）
python .cursor/skills/game-director-pro/scripts/concat_shots_to_final.py --input-dir 镜头目录 --output 成片.mp4
```

## 五、目录结构速览

| 路径 | 说明 |
|------|------|
| `SKILL.md` | 主技能说明，Agent 入口 |
| `README.md` | 本 skill 简介与安装概要 |
| `安装与使用说明.md` | 本文件，给安装者/其他人看 |
| `references/*.md` | 平台规则、API 绑定、公式与模板 |
| `scripts/*.py` | 首帧/尾帧生图、视频生成、成片合成等 |
| `.env.example` | 环境变量模板，复制为 .env 后填入 Key |
