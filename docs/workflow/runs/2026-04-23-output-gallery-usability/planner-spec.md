# Planner Spec - output gallery usability

> Run ID: `2026-04-23-output-gallery-usability`

## 背景

当前“服务端文件”页只是当前用户 output 目录的最近视频列表，支持预览、下载和保存到本地“我的成片”。随着即梦回补和服务端落盘数量增加，缺少搜索、筛选和用户级隐藏后，列表会快速失去可用性。

## 目标

1. 支持按关键字搜索服务端文件
2. 支持按来源与时间范围筛选
3. 支持“仅当前账号隐藏/恢复显示”，不删除物理文件、不影响即梦后台
4. 前端明确展示来源与当前视图边界

## 非目标

1. 不做物理删除
2. 不做历史即梦任务 owner 追溯修复
3. 不引入新的数据库表

## Approach

1. 后端新增 output gallery service，负责用户级隐藏索引读写与列表过滤
2. `GET /api/video/output-recent` 增加 `q/source/days/view` 查询参数
3. 新增隐藏/恢复接口，默认列表视图排除已隐藏项
4. 前端输出 tab 加搜索框、来源/时间/保存状态/显示视图筛选

## Acceptance Criteria

1. 当前账号可隐藏某个服务端文件，刷新后不再出现在默认列表
2. 切换到“已隐藏”视图后可看到并恢复该文件
3. 搜索与来源/时间筛选可生效
4. 已保存到本地“我的成片”的服务端文件有明确状态提示

## Risks

1. 用户隐藏索引如果只按 path 保存，需要保证 path 作为稳定主键可用
2. 前端筛选与服务端筛选混用时，需要避免重复过滤导致空列表误判
3. 不能影响既有视频预览与下载鉴权

## Test Matrix

1. 后端：过滤逻辑、隐藏/恢复逻辑
2. 前端：已保存状态判定、来源标签、筛选函数
3. 集成：列表读取、隐藏、恢复、保存到本地
4. 即梦回补：只允许当前账号已有归属证据的 submitId 进入回补候选
