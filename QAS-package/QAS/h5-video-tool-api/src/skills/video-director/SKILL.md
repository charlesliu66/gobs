# Video Director 视频导演技能

将用户的简单创意（如「10秒浪人打怪物」）转化为**可直接用于 Seedance/剪映平台**的视频生成 prompt。

## 工作流（严格执行）

```
前置：读取 storyboard-studio/SKILL.md + reference/seedance-rules.md
用户输入 → Step1 解析意图 → Step2 生成分镜脚本 → Step3 规则审核替换 → Step4 输出最终 prompt
```

**前置步骤**：执行前必须读取以下两个文件以获取分镜规范与审核规则：
1. `~/.cursor/skills/storyboard-studio/SKILL.md` — 镜头、构图、运镜等分镜规范
2. `~/.cursor/skills/video-director/reference/seedance-rules.md` — 平台禁用词与替换规则

---

## Step 1：解析用户意图

从用户输入中提取：

| 字段 | 提取规则 | 示例 |
|------|----------|------|
| **时长** | 数字+秒/分 | "10秒"→10，"30秒"→30 |
| **主体** | 人物/角色/场景 | 浪人、女孩、宇航员 |
| **动作/情节** | 核心事件 | 打怪物、跳舞、太空漫步 |
| **风格 hints** | 若有提及 | 暗黑、治愈、科幻 |

若未明确时长，默认 5–10 秒。若信息不足，可追问用户。

---

## Step 2：调用分镜技能生成脚本

**必须**先读取并遵循分镜规范：
- [~/.cursor/skills/storyboard-studio/SKILL.md](~/.cursor/skills/storyboard-studio/SKILL.md)（或通过 `/fenjing` 调用的 storyboard-studio skill）
- 镜头类型、构图、光线、运镜、时间码等按该技能规范执行

**必须**采用**分时间戳**的 prompt 格式，每个镜头明确起止时间：

```
【风格】风格锚点，视觉基调，色调。
【时长】N秒

[00:00-00:XX] 镜头1：名称（景别，运镜）。
具体场景与动作描述。含 @图片N 引用时按上传顺序映射。氛围/音效提示。

[00:XX-00:XX] 镜头2：名称（景别，运镜）。
...

[00:XX-00:XX] 镜头3：...

一致性约束。物理/色调说明。
```

### 分时间戳规范

| 要求 | 说明 |
|------|------|
| **时间码** | 每镜头必须写 `[00:00-00:XX]`，总时长与 `【时长】` 一致 |
| **镜头名** | 简明概括镜头功能，如「悬念·刀光」「对峙·双影」「交锋·收势」 |
| **景别+运镜** | 括号内标注，如「特写，缓慢推进」「全景，固定低角」 |
| **前段引人入胜** | 若用户要求「前3秒引人入胜」，首镜头需强悬念：特写、逆光、局部、剪影，避免平铺直叙 |

### 参考模板（浪人题材）

```
[00:00-00:03] 镜头1：悬念（刀刃特写，缓慢推进）。
雨滴打在刀刃上，寒光闪动。镜头推进至持刀近景，逆光剪影，雨幕朦胧。

[00:03-00:06] 镜头2：对峙（全景，固定低角）。
雨夜竹林，双方遥相对望。握刀，对方缓缓逼近。雨幕雾气。

[00:06-00:10] 镜头3：交锋与收势（中景跟拍）。
闪避挥刀，身形交错，刀光剑影。收刀而立，拉开距离。定格剪影。
```

**分镜要点**：镜头类型、构图、光线、运镜、时间码，按 storyboard-studio 规范执行。

---

## Step 3：Seedance 平台规则审核

**必须**对照 `reference/seedance-rules.md` 进行审核与替换。

逐句检查，将违规词替换为合规词，直至通过审核。若原创意过于暴力/敏感，可改为意境化、氛围化表述（如：对决→遥相对望，斩杀→收刀离去）。

---

## Step 4：输出最终 prompt

**若用户会上传素材**（video-pipeline 场景）：prompt 中**必须**用 `@图片1`、`@图片2` 等引用。按素材文件名**自动映射**：如文件夹有 `浪人.png`、`怪物.png`，则 prompt 中提及「浪人」→ 替换为 `@图片2`，提及「怪物」→ 替换为 `@图片1`。详见 [reference/seedance-rules.md](reference/seedance-rules.md)。

输出格式：

```markdown
## 最终 Prompt（可直接复制使用）

\`\`\`
[审核后的完整 prompt 文本，含 @图片N 引用]
\`\`\`

**参数建议**：duration=`N`, aspect_ratio=`16:9`, quality=`720p` 或 `1080p`
```

---

## 依赖

- **fenjing / storyboard-studio**：提供分镜与镜头语言规范
- **[reference/seedance-rules.md](reference/seedance-rules.md)**：平台内容审核词库，Step 3 必须读取并逐项执行

---

## 示例流程

**用户输入**：我想要做一个10秒的浪人打怪物的视频

**Step 1 解析**：时长=10秒，主体=浪人，动作=打怪物，风格=暗黑/剑戟片

**Step 2 分镜**：按 storyboard-studio 生成分时间戳 3 镜头脚本，如 `[00:00-00:03] 悬念`、`[00:03-00:06] 对峙`、`[00:06-00:10] 交锋收势`

**Step 3 审核**：按 seedance-rules 替换「怪物」→「神秘对手」、「斩杀」→「收刀」、「嘶吼」→「遥相对望」等

**Step 4 输出**：给出可直接复制的最终 prompt 及参数建议
