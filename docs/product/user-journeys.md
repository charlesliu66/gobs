# 用户路径

> Date: 2026-04-23  
> Purpose: 把平台能力收敛为用户能理解的三条路径。

## 一键成片

目标用户：新用户、运营同学、只想快速拿到视频的人。

Path:

1. 输入主题。
2. 选择风格、比例、时长。
3. 确认分镜。
4. 生成视频。
5. 查看结果。
6. 进入分发或剪辑。

Design rule: 不暴露 submit id、batch job 等工程概念，只展示用户态状态。

## 高级制片

目标用户：短剧创作者、需要角色/场景/分镜一致性的专业用户。

Path:

1. 剧本与故事弧。
2. 角色 / 场景设计。
3. 分镜表。
4. 分镜视频。
5. 放映室审片。
6. 导入剪辑器。

Design rule: 默认路径保持简单，专业能力放入“高级工具”。

## 分发运营

目标用户：负责账号矩阵、文案和发布复盘的运营同学。

Path:

1. 选择视频。
2. 选择平台账号。
3. 生成平台文案和标签。
4. 发布。
5. 查看结果和复盘。
6. 复用成功模板。

Design rule: 不混入制片配置，强调账号、平台、发布结果和复用。
