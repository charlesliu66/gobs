# TASK-B：自动打标引擎（体验版）

## 目标
打标结果"够用且可修正"，用户不用从零整理。

## 规则打标
- **比例**（9:16 / 16:9 / 1:1）
- **质量**（清晰度、水印风险）
- **时长**（短/中/长）
- **可用平台建议**（TikTok / YouTube）

## AI 打标
- **素材类型**（角色立绘 / 游戏截图 / 录屏 / 宣传图 / UI素材 等）
- **角色识别**（匹配项目已有角色卡）
- **场景识别**（战斗 / 日常 / 菜单 / 过场 等）
- **用途建议**（买量 / 社媒 / 版本宣发）

## 验收标准
- [ ] 每个标签有 confidence 值
- [ ] 低置信标签进入待确认队列
- [ ] 打标失败可单独重试，不阻塞整体导入任务
