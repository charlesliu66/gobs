## 上传：浮动按钮 → 底部上拉面板

点击右上角「📤 上传素材」→ 底部滑入一个上传面板（不离开画廊页面）：

```
                    ┌──────────────────────────────────────┐
                    │ ✕  上传素材                          │
                    ├──────────────────────────────────────┤
                    │                                      │
                    │   ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐   │
                    │   |  📂 拖入文件或点击选择          |   │
                    │   |  支持图片和视频，最多 500 个    |   │
                    │   └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘   │
                    │                                      │
                    │   [▓▓▓▓▓▓░░░░░░]  处理中 6/9        │
                    └──────────────────────────────────────┘
```

上传完成 → 面板自动关闭 → 画廊自动刷新 → 新素材出现在第一位。

## 待确认标签：横幅提醒 + 抽屉处理

当有待确认标签时，在筛选栏下方显示一条黄色横幅：

```
┌──────────────────────────────────────────────────────────────┐
│  ⚠  有 3 个 AI 标签需要确认，这将帮助更准确地检索素材  [立即处理 →]  │
└──────────────────────────────────────────────────────────────┘
```

点击「立即处理」→ 右侧滑出待确认队列（复用现有 `AssetReviewQueue` 逻辑）。

## 回答上版本的 5 个问题（我的推荐）

| 问题 | 推荐选择 | 理由 |
|---|---|---|
| 默认排序 | 最新上传在前 | 刚传完立刻能找到，符合直觉 |
| 卡片比例 | **正方形（1:1）** | 图片比例各异，正方形裁切最整齐；Figma/Canva 同款 |
| 多选批量操作 | **做**，只做批量删除 | 用户想清理旧素材时必须用；"批量用于生成"场景少，先不做 |
| 点卡片后 | **详情抽屉**（不跳页） | 留在画廊继续浏览，"用于生成"在抽屉里跳也来得及 |
| 分页方式 | **按需加载（Load more 按钮）** | 比无限滚动更可控，避免性能问题；每次加载 24 张 |
