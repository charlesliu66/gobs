# H5 视频创作工具 — 当前进度总结

**更新日期**: 2025-03-12

---

## 一、本机尝试（你本地跑起来验证）

### 1.1 启动步骤

```powershell
# 终端 1：后端
cd h5-video-tool-api
npm run dev

# 终端 2：前端
cd h5-video-tool
npm run dev
```

- 前端：http://localhost:5173
- 后端：http://localhost:3001（API 由 Vite 代理转发）

### 1.2 前置配置

| 配置项 | 说明 |
|--------|------|
| **Google OAuth** | 在 [Google Cloud Console](https://console.cloud.google.com) 创建 OAuth 2.0 客户端（Web 应用），启用 Drive API |
| **前端 .env** | `h5-video-tool/.env` 中设置 `VITE_GOOGLE_CLIENT_ID=你的客户端ID.apps.googleusercontent.com` |
| **网络** | 本机需能访问 Google（国内需代理） |

详见 `docs/README-google-drive.md`。

### 1.3 本机可验证的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| **素材管理 Tab** | ✅ 可测 | 粘贴 Drive 文件夹链接 → 连接 Drive → 验证权限 → 浏览子文件夹、图片/视频列表 |
| **图片预览** | ✅ 已修复 | 图片缩略图通过 `alt=media` 获取，支持个人盘和共享盘 |
| **视频生成 Tab** | ⚠️ 部分 | 输入 Prompt、提取关键词、选择素材 ✅；生成分镜、生成视频 ❌（API 未接） |
| **视频分发 Tab** | ⚠️ 壳子 | 页面在，发布 API 未接 |

### 1.4 本机验证重点

1. **Drive 连接**：点击「连接 Google Drive」→ 授权 → 能拿到 token
2. **素材库验证**：粘贴文件夹链接 → 点击「验证权限」→ 显示「素材库已就绪」
3. **素材库浏览**：进入子文件夹，能看到图片/视频列表
4. **图片预览**：列表中的图片能正常显示缩略图（不再空白）

## 二、分享给同事尝试

### 2.1 同事需要准备的环境

| 项目 | 说明 |
|------|------|
| **Node.js** | 建议 18+ |
| **代码** | 拉取 QAS 仓库，进入 `h5-video-tool` 和 `h5-video-tool-api` |
| **Google OAuth** | 使用同一项目的 OAuth 客户端，或同事自己创建（需配置 `http://localhost:5173` 为重定向 URI） |
| **网络** | 能访问 Google（国内需代理） |

### 2.2 同事启动方式（与本地相同）

```powershell
# 终端 1
cd h5-video-tool-api
npm install
npm run dev

# 终端 2
cd h5-video-tool
npm install
npm run dev
```

前端 `.env` 需配置 `VITE_GOOGLE_CLIENT_ID`。

### 2.3 分享给同事时可用的能力

| 能力 | 状态 | 同事能做什么 |
|------|------|--------------|
| **素材管理** | ✅ 可用 | 连接自己的 Drive、验证素材库、浏览文件夹、查看图片预览 |
| **视频生成（前半段）** | ✅ 可用 | 输入创意、提取关键词、选择素材、查看顺序映射 |
| **视频生成（后半段）** | ❌ 未接 | 生成分镜、生成视频 按钮无真实 API |
| **视频分发** | ❌ 未接 | 发布到社媒 无真实 API |

### 2.4 分享时的注意事项

1. **OAuth 客户端**：若用你的 OAuth 客户端，需在 Google Console 把同事的测试账号加入「测试用户」，否则会提示「应用未验证」
2. **后端需同网**：同事访问 `localhost:5173` 时，API 请求会走本机代理到 `localhost:3001`，所以前后端都在同事本机跑即可
3. **共享/部署**：若想让同事通过你的 IP 访问，需把 Vite 的 proxy 改为你的后端地址，或前后端分开部署（需处理 CORS、OAuth 重定向 URI 等）

### 2.5 建议的分享话术（可直接发给同事）

> **H5 视频创作工具试用**
>
> 1. 拉代码，进 `h5-video-tool` 和 `h5-video-tool-api` 分别 `npm install`
> 2. 在 `h5-video-tool` 下新建 `.env`，写入：`VITE_GOOGLE_CLIENT_ID=xxx`（找我要）
> 3. 两个终端分别跑 `npm run dev`，浏览器打开 http://localhost:5173
> 4. 进入「素材管理」→ 粘贴你的 Drive 素材文件夹链接 → 连接 Google Drive → 验证权限
> 5. 验证通过后可浏览文件夹、查看图片预览
> 6. 进入「视频生成」可体验：输入创意、提取关键词、选择素材（分镜和视频生成功能尚未接入）
