# 素材匹配 — 文件夹语义说明

**更新日期**: 2025-03-16

---

## 一、功能说明

当 prompt 提到「场景」「竹林」「雨夜」时，应优先从 **Scenario** 类文件夹（如 `Gold And Glory_Scenario`）取素材；提到「浪人」「怪物」时，从 **Character** 类文件夹取。文件夹名本身携带语义，用于提升匹配准确度。

---

## 二、配置：config/assets.json

```json
{
  "assetRoots": [
    "C:\\Users\\wei.liu\\Desktop\\cursor_try\\Ai test",
    "C:\\Users\\wei.liu\\Desktop\\cursor_try\\GNG Assets Library"
  ],
  "folderSemantics": {
    "Scenario": {
      "folderPatterns": ["Scenario", "场景", "Scene", "Background", "背景", "Environment", "环境"],
      "promptConcepts": ["场景", "背景", "环境", "竹林", "雨夜", "庭院", "森林", "沙漠", "场景图", "scene", "background", "environment"]
    },
    "Character": {
      "folderPatterns": ["Character", "角色", "人物", "NPC", "立绘"],
      "promptConcepts": ["角色", "人物", "浪人", "怪物", "战士", "主角", "对手", "character", "ronin", "warrior"]
    },
    "Weapon": {
      "folderPatterns": ["Weapon", "武器", "道具", "Item"],
      "promptConcepts": ["武器", "刀", "剑", "道具", "weapon", "sword", "katana"]
    },
    "Effect": {
      "folderPatterns": ["Effect", "特效", "粒子", "VFX"],
      "promptConcepts": ["特效", "粒子", "光效", "effect", "vfx"]
    }
  }
}
```

- **folderPatterns**：文件夹名包含任一 pattern 即视为该类型（如 `Gold And Glory_Scenario` 含 `Scenario` → Scenario 类型）
- **promptConcepts**：prompt 中出现这些词时，优先从对应类型文件夹搜索

---

## 三、匹配逻辑（video-pipeline 模式 A）

1. **提取关键词**：从 prompt 抽取实体词（浪人、怪物、竹林、雨夜 等）
2. **确定 folderHints**：若关键词命中某 `promptConcepts`，则加入对应 folderHints（如 竹林→Scenario，浪人→Character）
3. **递归扫描**：扫描 `assetRoots` 下所有文件（含 `GNG Assets Library`）
4. **文件夹过滤**：若 folderHints 非空，优先只从「路径中含 folderPatterns 的文件夹」内取文件
5. **文件名匹配**：在候选文件中，文件名包含关键词的优先
6. **排序**：文件夹语义匹配 > 文件名匹配数 > 文件名排序
7. **取前 N 个**：最多 5 个，对应 @图片1…@图片5

---

## 四、Drive 搜索（H5 工具）

- **promptPolish** 会输出 `folderHints`（如 `["Scenario","Character"]`）
- **Drive search** 收到 `folderHints` 时，只在该素材库根目录下「名称匹配 folderPatterns 的子文件夹」内搜索
- 例如：根目录为 `GNG Assets Library`，子文件夹 `Gold And Glory_Scenario` 名称含 `Scenario`，则 场景相关关键词 会只在该文件夹内搜

---

## 五、自定义配置

在 `config/assets.json` 中可增删：

- **assetRoots**：本地素材库根路径（video-pipeline 用）
- **folderSemantics**：新增类型如 `Prop`、`Landscape` 等，并设置 `folderPatterns` 与 `promptConcepts`
