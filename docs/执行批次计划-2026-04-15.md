## 三端同步标准流程（每批次末尾执行）

```
① npx tsc --noEmit          （本地编译检查，0 错误才能继续）
② git add / commit / push   （推送到 GitHub）
③ npm run build              （生成产物）
④ SFTP 上传 + pm2 restart   （部署到云端）
⑤ 验收检查                  （对照本批次验收清单逐项验证）
```

> ⚠️ 批次 0 例外：纯 SSH 服务器操作，无本地代码变动，跳过 ①②③，直接 ④⑤

**预计时间**：1-2 天

### 改动说明

**4-A：新建 resolver.ts**
```typescript
// src/infra/storage/resolver.ts
export type AssetType = 'video-output' | 'upload' | 'production' | 'quickfilm' | 'db' | 'editor-project'

export function resolvePath(type: AssetType, ...segments: string[]): string {
  const base = getApiDataDir()
  const roots: Record<AssetType, string> = {
    'video-output':    path.join(base, 'output'),
    'upload':          path.join(base, 'uploads'),
    'production':      path.join(base, 'output', 'production'),
    'quickfilm':       path.join(base, 'quickfilm'),
    'db':              path.join(base, 'db'),
    'editor-project':  path.join(base, 'editor-projects'),
  }
  return path.join(roots[type], ...segments)
}
```

**4-B：全局路径替换**
- `grep -r "path.join.*output\|getDefaultVideoOutput\|getUploads" src/` 找出所有硬编码路径
- 逐一替换为 `resolvePath(type, ...)`
- 旧的 `apiDataDir.ts` 中函数改为调用 resolver，保持向后兼容

## 批次 5：外部依赖状态机标准化（后端）

**涉及问题**：P1-G（状态机）、PM2 重启 17 次（稳定性）  
**改动文件**：

- `h5-video-tool-api/src/routes/videoDreamina.ts`（更新）
- `h5-video-tool-api/src/routes/videoKling.ts`（更新）
**预计时间**：1-2 天

### 改动说明

**5-A：统一状态枚举**
```typescript
// src/domain/job-status.ts
export type JobStatus = 'queued' | 'running' | 'succeeded' | 'failed' | 'timeout' | 'canceled'

export interface JobResult<T = unknown> {
  status: JobStatus
  data?: T
  errorCode?: 'DREAMINA_NOT_LOGGED_IN' | 'COMPASS_RATE_LIMIT' | 'TIMEOUT' | 'UNKNOWN'
  errorMessage?: string
  startedAt?: string
  endedAt?: string
}
```

**5-B：三类特殊场景标准化**
- Dreamina 未登录 → `{ status: 'failed', errorCode: 'DREAMINA_NOT_LOGGED_IN' }`
- Compass 429 → `{ status: 'failed', errorCode: 'COMPASS_RATE_LIMIT' }`
- 超时 → `{ status: 'timeout', errorCode: 'TIMEOUT' }`
- 前端根据 `errorCode` 显示具体提示，不再猜测状态

### 三端同步
```bash

git push origin main
cd h5-video-tool-api && npm run build
python scripts/deploy-api.py
```
