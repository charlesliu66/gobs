## 0. 一句话先理解 GOBS
## 0. GOBS In One Sentence

GOBS 是一个面向短视频内容团队的 AI 视频生产与分发平台。
GOBS is an AI video production and distribution platform built for short-form content teams.

它把“找素材、做创意、生成视频、精修剪辑、批量分发、查看结果”放在一套工作流里，适合运营和市场团队更快地产出 TikTok / 短视频内容。
It brings “finding assets, shaping creative ideas, generating videos, refining edits, distributing in batches, and tracking results” into one workflow, helping operations and marketing teams produce TikTok and short-form content faster.

如果只记一句话，可以这样介绍：
If you only remember one line, introduce it like this:

**GOBS = 用 AI 帮你把“创意想法”更快变成“可发布的短视频内容”。**
**GOBS = a platform that uses AI to turn creative ideas into publishable short-form video content faster.**

一句话总结：
One-line summary:

**这是从“出内容”走到“发出去”的最后一段。**
**This is the final step that turns finished content into published content.**

## 5. 推荐给运营 / 市场同学的使用习惯
## 5. Recommended Habits For Operations / Marketing Teammates

### 5.1 先从小题目开始，不要第一次就上复杂大案子
### 5.1 Start Small, Do Not Begin With A Large Complex Project

第一次试平台，推荐：
For the first trial, it is better to use:

- 一个简单人物
- One simple character
- 一个简单场景
- One simple scene
- 一个简单情绪方向
- One simple emotional direction

不推荐第一次就做：
Not recommended for a first trial:

- 超长剧情
- Very long narratives
- 角色特别多的复杂故事
- Complex stories with many characters
- 强依赖大量人工细节打磨的片子
- Projects that rely heavily on manual detail polishing

### 5.2 素材库要尽早用起来
### 5.2 Start Using The Asset Library Early

如果团队后面会反复做同一个系列、同一个角色、同一个 campaign 风格，素材库越早开始用，后面越省事。
If the team plans to repeatedly create content around the same series, character, or campaign style, the earlier you start using the Asset Library, the easier later production becomes.

最适合先沉淀进素材库的是：
The best things to store early in the Asset Library are:

- 主角色参考图
- Main character reference images
- 核心场景图
- Core scene references
- 产品视觉图
- Product visuals
- 常用视频素材
- Frequently used video assets

### 5.3 批量任务看板是必看的
### 5.3 The Batch Jobs Board Is A Must-Watch

不管是一键成片还是高级制片，只要进入批量生成，就建议看：
Whether you use Quick Film or Advanced Production, once you start batch generation, you should always check:

- “历史”
- “History”
- “批量任务看板”
- “Batch Jobs Board”

这样你能知道：
This gives you visibility into:

- 当前有没有在跑
- Whether tasks are currently running
- 是在排队还是生成中
- Whether they are queued or actively generating
- 哪一条成功 / 失败
- Which tasks succeeded or failed

### 5.4 分发不是黑盒，结果页里就能看
### 5.4 Distribution Is Not A Black Box, Results Are Visible On The Page

视频分发现在已经不是“点了发布就只能等”。
Video distribution is no longer a blind “click publish and just wait” experience.

发布后你可以直接在当前页看：
After publishing, you can directly see on the page:

- 每个账号是否提交成功
- Whether each account submission succeeded
- 是否运行中
- Whether it is still running
- 是否成功 / 失败
- Whether it succeeded or failed
- 失败原因
- Failure reasons
- 返回截图或链接
- Result screenshots or links

这对运营同学尤其重要，因为：
This is especially important for operations teammates because:

**你不需要再去猜“到底发没发出去”。**
**You no longer need to guess whether the content was actually published.**

## 7. 当前使用时需要提前告诉同事的注意事项
## 7. Important Notes To Tell Teammates Before They Start

### 7.1 不要把它当“一键完全替代人工”
### 7.1 Do Not Frame It As A One-Click Replacement For Human Work

更准确的定位是：
A more accurate positioning is:

**GOBS 是“帮你更快出内容基础和第一版”的平台，不是完全替代运营、创意、剪辑判断。**
**GOBS is a platform that helps teams produce the content foundation and the first version faster. It does not fully replace human judgment in operations, creativity, or editing.**

### 7.2 第一次建议用中文模式
### 7.2 For First-Time Local Users, Chinese Mode Is Still Recommended

虽然平台已经补了英文壳层，尤其是一键成片、视频分发和素材库主链路，但对于第一次上手的本地同事，仍建议先用中文模式，这样理解成本更低。
Although the platform already has English UI coverage, especially across Quick Film, Video Distribution, and the core Asset Library flows, I still recommend Chinese mode for first-time local teammates because it reduces learning friction.

### 7.3 先试短流程，再试重流程
### 7.3 Try The Lighter Flows First, Then The Heavier Ones

第一次建议先跑：
For a first trial, start with:

- 一键成片
- Quick Film
- 分发
- Distribution

不要一开始就直接跳到：
Do not start immediately with:

- 高级制片全流程
- The full Advanced Production flow
- 复杂剪辑 Agent 玩法
- More complex Editing Agent workflows

### 7.4 分发前确认账号权限和环境
### 7.4 Confirm Account Access And Environment Before Distribution

视频分发依赖 GeeLark 账号和环境配置。
Video distribution depends on GeeLark account access and environment setup.

如果某位同事发现自己选不到账号，通常不是“平台坏了”，而是账号权限还没分配到位。
If a teammate cannot see any available accounts, it usually does not mean the platform is broken. More often, their account permission setup is incomplete.
