# Planner Spec - production style ref asset picker

> Run: `2026-04-22-production-style-ref-asset-picker`
> Gate 1 产物

---

## 背景

高级制片 Step 0 的“参考图反解析”目前只支持本地上传。对已经把参考图维护在素材库中的用户来说，这会多出一轮“去素材库下载 → 再回高级制片上传”的重复动作，影响输入阶段的流畅度。

现有 `ProductionWizard.handleStyleRefFile` 已经承载了预览、上传到 `production/images`、风格反解析和摘要回填，因此本轮不新增后端能力，只补一个更顺手的入口。

---

## SR-01：Step 0 增加素材库选图入口

### 目标

在“参考图反解析”旁提供“从素材库选择”按钮，让用户在当前页面直接打开图片素材选择层。

### 方案

- `StepInput` 自己管理轻量状态
- 复用已有素材库 API：`listAssets`
- 只展示图片素材

### AC

| # | 场景 | 预期 |
|---|---|---|
| AC-1 | 进入 Step 0 | 能看到“从素材库选择”按钮 |
| AC-2 | 打开选择器 | 只展示图片素材，不展示视频/音频 |

---

## SR-02：选中素材后复用既有反解析链路

### 目标

素材库图片被选中后，不新增第二套解析逻辑，而是完全复用现有本地上传链路。

### 方案

- 将素材下载为 `Blob`
- 转成浏览器 `File`
- 调用既有 `onStyleRefFileChange(file)`

### AC

| # | 场景 | 预期 |
|---|---|---|
| AC-1 | 选择图片素材 | 关闭弹层并触发既有处理链路 |
| AC-2 | 风格反解析成功 | 预览和摘要照常更新 |
| AC-3 | 素材拉取失败 | 弹层内保留错误反馈，可重新选择 |

---
